# @sbkl/twicpics

TwicPics components for React, React Native (Expo), and Next.js. Provides context-aware optimization for images and videos with lazy loading, LQIP, optimized CLS, and more.

## Get Started

```sh
bun add @sbkl/twicpics
```

## Building

```sh
bun run build
```

## Testing

```sh
bun run test:common
bun run test:ui
bun run test:ui react    # only test UI for react
bun run test:ui next     # only test UI for next
```
